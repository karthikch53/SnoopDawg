SnoopDawg
=========
Searches the Twitter stream for the given query string. 

What it does?
Filter tweets based on the query string location within the given radius of 1000mi in and around Nagpur.
Geo code 21.1438 Latitude,79.0926 Longitude ,1000mi radius points to Nagpur.
